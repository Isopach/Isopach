### Focus

```text
read widely    break carefully    write clearly    ship useful artifacts
```

I keep notes and tools around the places where offensive security, defensive analysis, and practical engineering meet.

| Area | What shows up here |
| --- | --- |
| Security research | Bug bounty notes, web security, triage habits, proof-of-concept work |
| CTFs | Writeups, challenge notes, and reproducible learning paths |
| Tooling | Scripts and small utilities for making repetitive work less dull |
| Documentation | Public notes written for future-me and anyone walking the same path |
